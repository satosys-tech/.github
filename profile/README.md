# satosys-tech

  Open-source Lightning payments infrastructure.

  ## Projects

  | Repo | Description |
  |------|-------------|
  | [bitpos-core](https://github.com/satosys-tech/bitpos-core) | Self-hosted Lightning point-of-sale + Bolt Card wallet. One Docker container, your own keys. |
  | [bitpos-card-writer](https://github.com/satosys-tech/bitpos-card-writer) | Android app for programming and wiping NTAG 424 DNA Bolt Cards. |

  ## Stack

  TypeScript - React - Expo - PostgreSQL - Docker - Nostr Wallet Connect

  ## Get started

  ```bash
  curl -sSL https://bitpos.app/install.sh | bash
  ```
